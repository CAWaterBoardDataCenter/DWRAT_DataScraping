HOW TO RUN GIT on RSTUDIO
1) Install Git (not Git Extensions), from the Software Center
2) Create a folder called Github in your C:Users/<username>/Documents folder
3) Create an RSTUDIO Project file tied to your Github folder
4) Open a new terminal in RSTUDIO
5) Type the Git commands in the terminal


CHANGING FOLDER PATHS in GIT 
#Syntax
cd <folder path>

#Example
cd  C:\Users\palemi\Documents\Github\
#always use forward slashes in RStudio

CLONING WITH GIT
#Syntax#
	git clone --branch <branchname> <remote-repo-url>
	#our remote repository URL is https://github.com/paymanimal/DWRAT_DataScraping.git
	
#Example#
	git clone --branch Payman https://github.com/paymanimal/DWRAT_DataScraping.git
	
PUSHING And COMMITTING 
*this is how you to update the remote version of the repository; the remote version is hosted on GitHub
	#Syntax#
	git add * #this allows you to add all the files
	git commit -m "Message"
	git push origin <branchname>

	#Example#
	git add *
	git commit -m "Enter your comments about the code change"
	git push origin Manashi

FETCHING And PULLING
#this is how you update the local version of the repository with the latest code from GitHub

	#Syntax#
	git fetch
	git pull origin <branchname>
	
	#Example#
	git fetch
	git pull origin Payman
